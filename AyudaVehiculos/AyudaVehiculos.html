Informacion Sobre Vehiculos



Proceso para Agregar

  • Para que esta tenga un uso correcto, se deberá completar toda la información solicitada en cada uno de los campos de texto.
  • Se selecciona una marca de la lista desplegable.
  • Activo es para los vehiculos listos para utilizarse
  • Inactivo es para los vehiculos que no están operando en la empresa
  • Reparacion es para los vehiculos que están en mal estado
  • Para finalizar el proceso, se hará clic en el botón Agregar

    Proceso para Modificar

  • Para que este tenga un uso correcto, se deberá seleccionar un vehiculo de la tabla que contiene todos los vehiculos.
  • Automáticamente todos los campos de la parte superior se completarán.
  • Se deberá hacer clic en el botón modificar para habilitar la edición de campos.
  • Se debe modificar cada uno de los campos deseados para el vehiculo seleccionado, ya sea marca, color, entre otros
  • Aparecerán 2 botones, Aceptar y cancelar
  • Aceptar es para aplicar los cambios hechos al vehiculo, haga clic aquí para modificar.
  • Cancelar es para deshacer los cambios hechos y regresar al estado inicial

    Proceso para Eliminar

  • Para que este tenga un uso correcto, se deberá seleccionar un vehiculo de la tabla que contiene todos los vehiculos.
  • Automáticamente todos los campos de la parte superior se completarán.
  • Se deberá hacer clic en el botón eliminar para deshabilitar un vehiculo
  • Al hacer clic, aparecerá una ventana preguntándo por la confirmación de la acción
  • Aparecerán 2 botones, Aceptar y cancelar
  • Aceptar es para deshabilitar el vehículo seleccionado
  • Cancelar es para deshacer los cambios hechos y regresar al estado inicial

    Gestionar Modelos y Marcas

  • Este Módulo funciona para agregar, modificar y eliminar Modelos y Marcas, se accede haciendo clic en el botón blanco "Modelos/Marcas"

    Proceso para Agregar Marcas

  • Para que este funcione correctamente, se deberá escribir un nombre para la marca a agregar en el campo de texto que dice Marca
  • Se deberá hacer clic en el botón Agregar, automáticamente la marca aparecerá en la tabla de la parte inferior de la ventana.

    Proceso para Modificar Marcas

  • Para que este tenga un uso correcto, se deberá seleccionar una marca de la tabla que contiene todas las marcas.
  • Automáticamente el campos de la parte superior se completará.
  • Se deberá hacer clic en el botón modificar para habilitar la edición de campos.
  • Aceptar es para aplicar los cambios hechos a la marca, haga clic aquí para modificar.
  • Cancelar es para deshacer los cambios hechos y regresar al estado inicial

    Proceso para Agregar Modelos

  • Se tendrá que escribir un nombre en el campo de texto que dice Modelo, este será el nombre del modelo a ingresar.
  • Seleccionamos, desde la lista que dice Marca, una marca existente para el modelo a agregar.
  • Se hace clic en el botón agregar para insertar el nuevo modelo creado, automáticamente aparecerá en la lista de la derecha.

    Proceso para Modificar Modelos

  • Para que este funcione correctamente, se deberá seleccionar una marca de la tabla
    izquierda, automáticamente, todos los modelos de esa marca aparecerán en la tabla derecha
  • Se deberá hacer clic en el botón modificar para habilitar la edición de campos.
  • Aceptar es para aplicar los cambios hechos al modelo, haga clic aquí para modificar.
  • Cancelar es para deshacer los cambios hechos y regresar al estado inicial
